# Disease Prediction System using Machine Learning

This project predicts diseases based on symptoms using a Decision Tree Machine Learning model.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Features

* Symptom based disease prediction
* Interactive user input
* Machine learning model training
* Simple dataset analysis

## Example

Input symptoms:
Fever = 1
Cough = 1
Headache = 1
Vomiting = 0

Output:
Predicted Disease = Flu
